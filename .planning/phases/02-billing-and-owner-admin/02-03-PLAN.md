    1. Create apps/api/src/routes/billing/create-checkout-intent.ts:
       - Export createCheckoutIntentRoute as Fastify plugin
       - POST /create-checkout-intent — authenticated (in protected scope)
       - Body: { priceId: string } validated with Zod
       - Handler:
         a. Get tenantId from request.user
         b. Query TenantSubscription for stripeCustomerId
         c. If no stripeCustomerId: create Stripe customer via stripe.customers.create({ metadata: { tenantId } }), update TenantSubscription
         d. Create subscription: stripe.subscriptions.create({ customer: stripeCustomerId, items: [{ price: priceId }], payment_behavior: 'default_incomplete', payment_settings: { save_default_payment_method: 'on_subscription' }, expand: ['latest_invoice.payment_intent'] })
         e. Update TenantSubscription with stripeSubscriptionId
         f. Extract clientSecret from subscription.latest_invoice.payment_intent.client_secret
         g. Return { clientSecret, subscriptionId: subscription.id }

    2. Create apps/api/src/routes/billing/sync-checkout.ts:
       - POST /sync-checkout — authenticated
       - Handler:
         a. Get tenantId, query TenantSubscription for stripeSubscriptionId
         b. Call stripe.subscriptions.retrieve(stripeSubscriptionId) directly from Stripe
         c. Update TenantSubscription: status, currentPeriodStart, currentPeriodEnd
         d. Invalidate Redis plan cache: redis.del(`plan:${tenantId}`)
         e. Return { status: stripeSub.status, plan: planTier }

    3. Create apps/api/src/routes/billing/invoices.ts:
       - GET /invoices — authenticated
       - Query stripe.invoices.list({ customer: stripeCustomerId, limit: 20 })
       - Return mapped array: { id, number, status, amountDue, amountPaid, currency, created, hostedInvoiceUrl, pdfUrl }

    4. Create apps/api/src/routes/billing/payment-method.ts:
       - POST /update-payment-method — authenticated
       - Body: { paymentMethodId: string }
       - Attach PM to customer: stripe.paymentMethods.attach(paymentMethodId, { customer: stripeCustomerId })
       - Set as default: stripe.customers.update(stripeCustomerId, { invoice_settings: { default_payment_method: paymentMethodId } })
       - Return { success: true }

    5. Create apps/api/src/routes/billing/cancel.ts:
       - POST /cancel — authenticated
       - Set cancel_at_period_end: stripe.subscriptions.update(stripeSubscriptionId, { cancel_at_period_end: true })
       - Update TenantSubscription: cancelAtPeriodEnd = true
       - Return { cancelAtPeriodEnd: true, currentPeriodEnd }

    6. Update apps/api/src/routes/billing/index.ts:
       - Register all new routes under /billing prefix
       - The webhook route stays PUBLIC (already registered)
       - All other billing routes (create-checkout-intent, sync-checkout, invoices, payment-method, cancel) are AUTHENTICATED

    7. Update apps/api/src/server.ts:

    1. Create apps/worker/src/workers/trial-expiry.ts:
       - Export trialExpiryWorker = new Worker(QUEUE_NAMES.TRIAL_EXPIRY, handler, { connection: bullmqConnection })
       - Handler processes two scenarios:

       a. Dunning check (trial-3d warning):
          - Query TenantSubscription WHERE status = 'TRIALING' AND trialEnd BETWEEN now AND now+3days
          - For each: enqueue email notification to emailNotificationQueue with type 'trial-expiring' (template content at Claude's discretion)
          - Log: "Sent trial expiry warning to tenant ${tenantId}"

       b. Trial expiry (suspension):
          - Query TenantSubscription WHERE status = 'TRIALING' AND trialEnd < now
          - For each:
            - Update TenantSubscription: status = 'SUSPENDED'
            - Update Tenant: status = 'SUSPENDED', suspendedAt = now
            - Invalidate Redis plan cache: redis.del(`plan:${tenantId}`)
            - Enqueue email notification with type 'trial-expired'
          - Log: "Suspended tenant ${tenantId} — trial expired"

    2. Update apps/worker/src/index.ts:
       - Import trialExpiryWorker
       - Add repeatable job: trialExpiryQueue.add('daily-trial-check', {}, { repeat: { pattern: '0 6 * * *' } }) (6 AM daily)
