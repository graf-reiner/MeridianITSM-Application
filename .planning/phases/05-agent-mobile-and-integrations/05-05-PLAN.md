1. **MeridianApiClient** — Create `apps/inventory-agent/src/InvAgent.Http/MeridianApiClient.cs`:

   NuGet packages: `Microsoft.Extensions.Http.Resilience` (includes Polly 8.x), `System.Text.Json`.

   ```csharp
   namespace InvAgent.Http;
   using InvAgent.Config;
   using InvAgent.Models;

   public class MeridianApiClient
   {
       private readonly HttpClient _http;
       private readonly AgentConfig _config;

       public MeridianApiClient(HttpClient http, AgentConfig config)
       {
           _http = http;
           _config = config;
           _http.BaseAddress = new Uri(config.ServerUrl);
           if (!string.IsNullOrEmpty(config.AgentKey))
               _http.DefaultRequestHeaders.Add("Authorization", $"AgentKey {config.AgentKey}");
       }

       public async Task<EnrollmentResult?> EnrollAsync(string token, string hostname, string platform, string agentVersion, CancellationToken ct);
       public async Task SendHeartbeatAsync(HeartbeatPayload payload, CancellationToken ct);
       public async Task<string?> SubmitInventoryAsync(InventoryPayload payload, CancellationToken ct);
       public async Task SubmitCmdbSyncAsync(InventoryPayload payload, CancellationToken ct);
   }
   ```

   Register in DI with Polly resilience:
   ```csharp
   builder.Services.AddHttpClient<MeridianApiClient>()
       .AddResilienceHandler("meridian-retry", pipeline =>
       {
           pipeline.AddRetry(new HttpRetryStrategyOptions
           {
               MaxRetryAttempts = 10,
               Delay = TimeSpan.FromSeconds(30),
               BackoffType = DelayBackoffType.Exponential,
               UseJitter = true,
           });
           pipeline.AddCircuitBreaker(new HttpCircuitBreakerStrategyOptions
           {
               SamplingDuration = TimeSpan.FromMinutes(2),
               FailureRatio = 0.5,
               MinimumThroughput = 3,
           });
       });
   ```

   If HttpProxy configured, set `HttpClientHandler.Proxy` via `WebProxy`.

3. **AgentWorker** — Create `apps/inventory-agent/src/InvAgent.Worker/AgentWorker.cs`:

   Extends `BackgroundService`:
   ```csharp
   public class AgentWorker : BackgroundService
   {
       private readonly ICollector _collector;
       private readonly MeridianApiClient _api;
       private readonly LocalQueue _queue;
       private readonly AgentConfig _config;
       private readonly PrivacyFilter _privacy;
       // Constructor injection
   }
   ```

   `ExecuteAsync` logic:
   - On startup: attempt to flush any queued items from previous session
   - If AgentKey is null and EnrollmentToken is set: call `_api.EnrollAsync()`, store agentKey in config file, set on HttpClient
   - Start two periodic timers:
     - Heartbeat: every `HeartbeatIntervalSeconds` (default 300s = 5min)
     - Inventory: every `InventoryIntervalSeconds` (default 14400s = 4hr)
   - Heartbeat tick: call `_api.SendHeartbeatAsync()`, catch exceptions (log, don't crash)
   - Inventory tick:
     1. Call `_collector.CollectAsync(ct)`
     2. Apply `PrivacyFilter.Apply(payload, _config.PrivacyTier)`
     3. Try `_api.SubmitInventoryAsync(payload)` and `_api.SubmitCmdbSyncAsync(payload)`
     4. On failure: `_queue.Enqueue("inventory", JsonSerializer.Serialize(payload))`
     5. On success: attempt `_queue.FlushAsync(_api)` to drain any queued items
   - If `--run-once` CLI flag: collect once, submit, exit
