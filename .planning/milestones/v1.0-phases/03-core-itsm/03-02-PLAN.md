2. **Create `apps/api/src/services/sla.service.ts`**:

   Import `toZonedTime`, `fromZonedTime` from `date-fns-tz` and `addMinutes` from `date-fns`.

   **Priority-to-field mapping**:
   ```typescript
   type Priority = 'LOW' | 'MEDIUM' | 'HIGH' | 'CRITICAL';
   const PRIORITY_MAP: Record<Priority, { response: string; resolution: string }> = {
     CRITICAL: { response: 'p1ResponseMinutes', resolution: 'p1ResolutionMinutes' },
     HIGH:     { response: 'p2ResponseMinutes', resolution: 'p2ResolutionMinutes' },
     MEDIUM:   { response: 'p3ResponseMinutes', resolution: 'p3ResolutionMinutes' },
     LOW:      { response: 'p4ResponseMinutes', resolution: 'p4ResolutionMinutes' },
   };
   ```

   **Export `function getResponseMinutes(sla, priority)`**: Returns the response target in minutes for the given priority using PRIORITY_MAP.

   **Export `function getResolutionMinutes(sla, priority)`**: Returns the resolution target in minutes for the given priority using PRIORITY_MAP.

   **Export `function calculateBreachAt(startTime: Date, targetMinutes: number, sla: SlaPolicy): Date`**:
   - If `!sla.businessHours`, return `addMinutes(startTime, targetMinutes)`
   - Extract timezone (`sla.timezone ?? 'UTC'`), businessHoursStart (default '09:00'), businessHoursEnd (default '17:00'), businessDays (default [1,2,3,4,5])
   - Convert startTime to zoned time via `toZonedTime(startTime, tz)`
   - Walk forward through business hours: for each business day, calculate available minutes, subtract from remaining
   - Skip non-business days entirely
   - If current time is before business hours start, snap to start
   - If current time is after business hours end, advance to next business day start
   - Return `fromZonedTime(resultZoned, tz)` to convert back to UTC
   - Implementation follows the exact algorithm from 03-RESEARCH.md Pattern 1

   **Export `function calculateResponseAt(startTime: Date, sla, priority: Priority): Date`**:
   - Calls `calculateBreachAt(startTime, getResponseMinutes(sla, priority), sla)`

   **Export `function calculateResolutionBreachAt(startTime: Date, sla, priority: Priority): Date`**:
   - Calls `calculateBreachAt(startTime, getResolutionMinutes(sla, priority), sla)`

   **Export `function getElapsedPercentage(startTime: Date, breachAt: Date): number`**:
   - `elapsed = Date.now() - startTime.getTime()`
   - `total = breachAt.getTime() - startTime.getTime()`
   - Returns `Math.round((elapsed / total) * 100)`, clamped to 0 minimum

   **Export `function getSlaStatus(percentage: number): 'OK' | 'WARNING' | 'CRITICAL' | 'BREACHED'`**:
   - `>= 100` returns `'BREACHED'`
   - `>= 90` returns `'CRITICAL'`
   - `>= 75` returns `'WARNING'`
   - Otherwise `'OK'`
