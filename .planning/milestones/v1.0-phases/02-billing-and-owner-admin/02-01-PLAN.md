       - Export webhookRoute as a Fastify plugin
       - CRITICAL: Use Fastify's route-level `preParsing` hook to capture raw body as Buffer. Do NOT use global addContentTypeParser (it breaks all other routes). Pattern:
         ```typescript
         fastify.post('/billing/webhook', {
           config: { rawBody: true },
           preParsing: async (request, _reply, payload) => {
             // Collect raw body chunks into Buffer
             const chunks: Buffer[] = [];
             for await (const chunk of payload) {
               chunks.push(typeof chunk === 'string' ? Buffer.from(chunk) : chunk);
             }
             request.rawBody = Buffer.concat(chunks);
             // Return a readable stream of the same data for Fastify's parser
             const { Readable } = await import('node:stream');
             return Readable.from(request.rawBody);
           },
         }, async (request, reply) => { ... });
         ```
       - Declare rawBody on FastifyRequest via module augmentation:
         ```typescript
         declare module 'fastify' {
           interface FastifyRequest { rawBody?: Buffer; }
         }
         ```
       - In the handler: get `stripe-signature` header, call `stripe.webhooks.constructEvent(request.rawBody!, sig, process.env.STRIPE_WEBHOOK_SECRET!)` inside try/catch
       - On verification failure: return 400 `{ error: 'Webhook signature verification failed' }`
       - On success: import Queue from bullmq and stripeWebhookQueue from worker definitions OR create a local queue instance. Enqueue: `await stripeWebhookQueue.add('stripe-event', { eventId: event.id, eventType: event.type, payload: event })`
